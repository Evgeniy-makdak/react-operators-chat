# Security Policy

## Supported versions

This repository is maintained on a best-effort basis as a reference implementation. Security fixes are applied to `main`.

## Reporting a vulnerability

If you discover a security issue, please do not publish exploit details in public issues.

Report privately through GitHub security advisory (preferred) or contact the maintainer:
- GitHub: [Evgeniy-makdak](https://github.com/Evgeniy-makdak)

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Suggested mitigation (if available)

## Notes

Because this project documents auth and WebSocket patterns, validate token transport and CORS settings carefully in your production environment.
